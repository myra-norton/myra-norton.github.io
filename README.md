# the how and why
i do a lot of random little side projects that never get to see the light of day - they're not "cool enough" for instagram, "funny enough" for twitter, or even just "enough" for me to want others to see

this is a place for me to compile all of those things

i've always wanted to build a website, but i didn't want to use wix or squarespace because i wanted to understand the guts of how websites work. over winter break, i found an outdated (but free) coursera course on HTML, CSS, and Javascript and spent a couple days in a coffee shop back home designing and creating this lil guy.

i haven't pushed everything yet because the perfectionist in me doesn't think its "enough" yet, but i hope you enjoy what i've put up :)
